# Drupal Skills Index

This file lists all available skills for hook-based detection and activation.

## Entity API Skills

### drupal-entity-arch
**File:** `entity-api/entity-architecture.md`
**Triggers:** entity architecture, content entity vs config entity, should I use bundles, revisionable, translatable, entity type decision
**Use when:** Making architectural decisions about entity types, choosing between content and config entities, deciding on bundles/revisions/translation support.

### drupal-entity-create
**File:** `entity-api/content-entity-creation.md`
**Triggers:** create custom entity, scaffold entity, entity handlers, entity forms, list builder, access handler, content entity type
**Use when:** Creating a new custom content entity from scratch, implementing entity handlers, setting up entity forms and routing.

### drupal-entity-ops
**File:** `entity-api/entity-operations.md`
**Triggers:** load entity, entity query, entityQuery, save entity, delete entity, render entity, entity CRUD
**Use when:** Working with existing entities - loading, querying, creating, updating, deleting, or rendering entities.
